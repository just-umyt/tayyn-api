# REST-API на Golang

Простенький апишка на го по архитектуре дяди Боб. Если есть ошибки или еще коечто, я готов исправлять.
Инструменты:
- Фреймворк fiber
- gorm.io
- mysql
- jwt
- .env(для config)
Действии api:
- Регистрация и Логин пользователей;
- Создание блогов
- middleware(те кто авторизовались)
